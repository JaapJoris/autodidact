How quizzes work

Each course can have multiple quizzes. When the student clicks the "Take the quiz" button, a quiz is randomly selected from these quizzes. This page shows the overview of all available quizzes per course. Click a quiz below to edit its questions, or add a new quiz using the button on the right.
